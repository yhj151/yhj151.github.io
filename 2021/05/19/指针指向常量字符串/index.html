  • 指针指向常量字符串

​ 当相同常量字符串放在内存中时，因为其不能改变，又相同，为了节省内存，故只需存一份，因此p1和p2都指向的是常量字符串的首地址，所以p1==p2.

#include<stdio.h>

int main()
{
	char *p1 = "hello bit";
	char *p2 = "hello bit";

	if (p1 == p2)
	{
		printf("hehe\n");
	}
	else
	{
		printf("haha\n");
	}
	
	return 0;

}

结果：

hehe
